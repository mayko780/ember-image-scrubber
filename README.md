## Installation

`ember install ember-image-scrubber`

## Usage

```
{{#image-scrubber}}
  <img src="http://lorempixel.com/200/200/people/1" />
  <img src="http://lorempixel.com/200/200/people/2" />
  <img src="http://lorempixel.com/200/200/people/3" />
{{/image-scrubber}}
```
